# Referee Dashboard

## Purpose

Provide a clear view of the referee's development over time.

The dashboard should transform game observations into actionable insights.

---

# Main Menu

- Dashboard
- New Game
- History
- Development
- Profile

---

# Dashboard Home

## Summary

Display:

- Total games worked
- Total self-assessments
- Total observations received
- Current season statistics

---

## Strengths

Display recurring positive observations.

Examples:

- Communication
- Game Control
- Mechanics
- Teamwork

Goal:

Help referees identify their strongest areas.

---

## Areas for Development

Display recurring observations that require attention.

Examples:

- Verticality
- Block/Charge
- Rotations
- Preventive Officiating

Goal:

Help referees focus their development efforts.

---

## Development Focus

Display the category that appears most frequently in recent observations.

Example:

Current Development Focus:
Verticality

Observed in 5 of the last 8 games.

---

## Recent Activity

Display:

- Latest games
- Latest observations
- Latest self-assessments
