# User Flow

## Before the Game

- Game is assigned
- Referees are defined

## During the Game

- Referees work the game
- Coordinator observes

## After the Game

### Referee

- Completes self-assessment

### Coordinator

- Registers observations

### Feedback Session

- Discussion occurs
- Development points are recorded

## Long Term

- Referee accesses history
- Platform identifies patterns
- Development is monitored
