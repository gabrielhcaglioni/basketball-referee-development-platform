# Screens

## Referee Dashboard

Purpose:

Allow the referee to monitor personal development.

Information:

- Total games
- Self-assessments completed
- Main recurring categories
- Recent observations
- Development history

---

## Self Assessment Screen

Purpose:

Allow post-game reflection.

Information:

- Game information
- Questionnaire
- Personal comments

---

## Referee History

Purpose:

View development over time.

Information:

- Games worked
- Observations received
- Self-assessments
- Patterns identified

---

## Coordinator Dashboard

Purpose:

Monitor referee development.

Information:

- Referees
- Recent assessments
- Development trends

---

## Observation Registration

Purpose:

Register feedback.

Information:

- Category
- Subcategory
- Description
- Development recommendation

---

## Feedback Session

Purpose:

Record post-game discussion.

Information:

- Summary
- Positive points
- Development points
- Agreed actions
